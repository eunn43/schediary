## 일정 + 일기 서비스

<br/>

일정과 일기를 카드 형식으로 보관하는 일기 서비스입니다.

- 개인프로젝트로 진행했습니다.
- next.js의 동적 라우팅을 사용했습니다.
- react-calendar와 react-colorful 라이브러리를 사용했습니다.
- open api를 사용하여 날씨 정보를 확인할 수 있습니다.
